# Clinical LLM Evaluation

A portfolio project focused on evaluating Large Language Models (LLMs) in clinical medicine.

## Goals

- Assess clinical accuracy
- Assess clinical safety
- Compare different LLMs
- Identify hallucinations and missing information

## Models

- ChatGPT
- Gemini
- Claude

## Evaluation Domains

- Internal Medicine
- Emergency Medicine
- Endocrinology
- Occupational Medicine
